# Data Availability Statement

The supplementary materials, sample images, implementation files, and experimental outputs associated with this study are publicly available in this GitHub repository.

The object detection component utilizes YOLOv12 models trained on the publicly available COCO dataset. Therefore, the original dataset is referenced through its official source and is not redistributed in this repository.

This repository provides:

* Dataset source information
* Sample test images
* Object detection outputs
* Depth estimation outputs
* Localization examples
* Contextual reasoning examples
* Core implementation files

Additional information may be provided by the corresponding author upon reasonable request.
