# Deep Learning-Based Framework for Object Detection, Localization, and Contextual Awareness for Visually Impaired Individuals

## Overview

This repository contains supplementary materials, sample data, implementation files, and experimental outputs associated with the research paper:

"Deep Learning-Based Framework for Object Detection, Localization, and Contextual Awareness for Visually Impaired Individuals"

The framework integrates:

* YOLOv12 for object detection
* MiDaS for monocular depth estimation
* ORB-based Visual Odometry (ORB-VO) for localization
* LLaVA Vision-Language Model for contextual reasoning

## Repository Contents

* Dataset source information
* Sample input images
* Sample experimental outputs
* Core implementation files
* Supplementary materials

## System Components

### Object Detection

YOLOv12 pretrained model trained on the COCO dataset.

### Depth Estimation

MiDaS monocular depth estimation model.

### Localization

ORB-based Visual Odometry.

### Contextual Reasoning

LLaVA Vision-Language Model.
