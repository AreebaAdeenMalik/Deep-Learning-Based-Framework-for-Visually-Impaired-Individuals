This is a unified deep learning-based framework that enhances real-time environmental perception by integrating object detection, distance estimation, user localization, and contextual reasoning into a single system. This repository contains the data and supplementary materials associated with the paper:

# Deep Learning-Based Framework for Object Detection, Localization, and Contextual Awareness for Visually Impaired Individuals

# Object Detection:
YOLOv12 model trained on the COCO dataset.

# Depth Estimation:
MiDaS.

# Localization:
ORB-based Visual Odometry.

# Contextual Reasoning:
LLaVA Vision-Language Model.

This repository provides:
- Dataset source links
- Sample test images
- Sample output results
- Supplementary materials used for evaluation
