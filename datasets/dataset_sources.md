# Dataset Sources

## COCO Dataset

Official Website:
https://cocodataset.org

Description:
The object detection and depth estimation component utilizes YOLOv12 weights originally trained on the COCO dataset.

The COCO dataset contains:

* 330,000+ images
* 200,000+ labeled images
* 1.5 million object annotations
* 80 object categories

The dataset is publicly available and is not redistributed in this repository.
